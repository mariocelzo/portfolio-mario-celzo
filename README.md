# Mario Celzo - Personal Portfolio

A sleek, Apple-style personal portfolio built with React, Tailwind CSS, and Framer Motion. 

## Features
- **Modern Apple-Style Aesthetics:** Clean typography, glassmorphism, soft drop shadows, and premium layout structure.
- **Dynamic Theming:** Seamless Dark and Light mode, dynamically updating accents (Baby Blue on Navy vs. Navy on Light Azure).
- **Smooth Animations:** Integrated scroll reveals and micro-interactions powered by `motion/react`.
- **Responsive Design:** Fully fluid across mobile, tablet, and desktop views.
- **Structured Sections:** 
  - Hero (Profile & Intro)
  - About Me
  - Experience & Education (Detailed timeline-style cards)
  - Skills (Categorized tech stack)
  - Projects (Grid of featured work with live Vercel links)
  - Contact Form

## Tech Stack
- React 18
- Tailwind CSS v4
- Framer Motion
- Lucide React (Icons)
- Vercel (Ready for deployment)

## Setup & Local Development

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd <your-repo-name>
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

## Deployment
This project is fully optimized for Vercel. Connect your repository to Vercel and it will automatically build and deploy your application.
